# 📓 랜덤 주제 일기 작성 서비스 (CapstonDesign)

## 📢 프로젝트 소개
**CapstonDesign**은 어떤 글을 써야 할지 막막한 분들을 위해 **다양한 랜덤 주제를 제공해 주는 일기 작성 서비스**입니다. 
사용자는 제공된 흥미로운 주제를 바탕으로 자신의 생각과 감정, 일상을 보다 쉽게 기록할 수 있습니다.

## 🛠 현재 기술 스택
- **Framework & Library:** React, Next.js
- **Package Manager:** npm

## 🚀 향후 마이그레이션 계획 (예정)
현재 프로젝트는 향후 더 높은 안정성과 성능 최적화, 그리고 개발자 경험(DX) 향상을 위해 다음과 같은 마이그레이션을 계획하고 있습니다.
- **TypeScript 도입:** 정적 타입 검사를 통한 오류 방지 및 유지보수성 향상
- **Next.js 전면 고도화:** 최신 Next.js 환경으로 완벽히 이전하여 SSR/SSG를 통한 성능 개선 및 SEO 최적화

## ⚙️ 설치 및 실행 방법
이 프로젝트를 로컬 환경에서 실행하려면 아래의 과정을 따라주세요.

```bash
# 1. 저장소 클론
git clone [https://github.com/jinkong9/CapstonDesign.git](https://github.com/jinkong9/CapstonDesign.git)

# 2. 프로젝트 디렉토리로 이동
cd CapstonDesign

# 3. 의존성 패키지 설치
npm install

# 4. 개발 서버 실행
npm run dev
